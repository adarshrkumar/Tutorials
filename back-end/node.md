# Node.js

## What is Node.js?

Node.js is a JavaScript runtime built on Chrome's V8 engine that allows you to run JavaScript on the server side. It enables building scalable network applications, APIs, and backend services using JavaScript. Node.js uses an event-driven, non-blocking I/O model.
