# npm (Node Package Manager)

## What is npm?

npm is the default package manager for Node.js. It allows you to install, share, and manage dependencies (libraries and tools) in your JavaScript projects. npm hosts hundreds of thousands of reusable code packages and helps manage project dependencies through package.json.
