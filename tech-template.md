## HTML (HyperText Markup Language)

### What is HTML?
HTML is the standard markup language for creating web pages. It provides the structure and content of a webpage using elements represented by tags. HTML defines headings, paragraphs, links, images, and other content that browsers display.

## CSS (Cascading Style Sheets)

### What is CSS?
CSS is a styling language used to describe the presentation of HTML documents. It controls layout, colors, fonts, spacing, and responsive design. CSS allows you to separate content from design, making websites more maintainable and visually appealing.

### What is JavaScript?

## Node.js

### What is Node.js?
Node.js is a JavaScript runtime built on Chrome's V8 engine that allows you to run JavaScript on the server side. It enables building scalable network applications, APIs, and backend services using JavaScript. Node.js uses an event-driven, non-blocking I/O model.

## npm (Node Package Manager)

### What is npm?
npm is the default package manager for Node.js. It allows you to install, share, and manage dependencies (libraries and tools) in your JavaScript projects. npm hosts hundreds of thousands of reusable code packages and helps manage project dependencies through package.json.

## MongoDB

### What is MongoDB?
MongoDB is a NoSQL document database that stores data in flexible, JSON-like documents. Unlike traditional relational databases, MongoDB doesn't require a predefined schema. It's designed for scalability and flexibility, making it popular for modern web applications.

## TypeScript

### What is TypeScript?
TypeScript is a superset of JavaScript that adds static typing. It compiles to plain JavaScript and helps catch errors during development through type checking. TypeScript improves code quality, maintainability, and provides better tooling support with autocomplete and refactoring.
